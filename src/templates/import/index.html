Importar Registros de Horas Extras

Instruções

Utilize esta página para importar registros de horas extras a partir de uma planilha Excel (.xlsx).

A planilha deve conter as seguintes colunas:

  • Nome Funcionário: Nome completo do funcionário (deve existir no sistema)
  • Data: Data no formato DD/MM/YY
  • Hora Extra Inicial: Horário de início no formato HH:MM
  • Hora Extra Final: Horário de término no formato HH:MM
Exemplo de planilha:
Nome Funcionário Data Hora Extra Inicial Hora Extra Final
João Silva 01/03/25 03:00 06:00
João Silva 01/03/25 19:00 20:00
Maria Santos 02/03/25 22:00 23:59
