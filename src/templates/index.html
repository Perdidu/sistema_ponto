Sistema de Controle de Ponto

Bem-vindo ao Sistema de Controle de Ponto

Este sistema permite gerenciar o controle de ponto de funcionários conforme as leis trabalhistas brasileiras.

Funcionalidades principais:

  • Cadastro de funcionários
  • Registro de ponto (entrada, saída, almoço)
  • Cadastro de feriados
  • Cálculo automático de horas extras, adicional noturno e DSR
  • Relatórios financeiros baseados no salário
